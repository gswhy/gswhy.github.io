脚注

脚注1是文档中用于添加补充说明或引用来源的一种机制。

脚注的特点

脚注具有以下特点2：

  • 自动编号3
  • 支持双向跳转4
  • 语法简单直观5

使用场景

脚注适用于以下场景6 7：

  1. 技术博客
  2. 学术笔记
  3. 参考文献管理

相册功能

要创建一个相册，只需要将多张图片放在同一行（或同一个段落）中。

语法 

1
2
3

![图片 1](image1.jpg)  ![图片 2](image2.jpg) 

![图片 3](image3.jpg)  ![图片 4](image4.jpg)

图片之间应该有两个空格，以确保它们在 Markdown 中保持在同一行。

参考文献示例 (脚注)

  1. 脚注是出现在页面底部的注释，用于对正文内容进行补充说明或标注引用来源。 ↩︎

  2. 本示例来源于 Markdown 脚注功能测试文档。 ↩︎

  3. 脚注的编号是自动生成的，无需手动维护序号。 ↩︎

  4. 点击正文中的脚注标记可以跳转到对应的脚注内容，点击脚注后的返回箭头可以跳回原文位置。 ↩︎

  5. 脚注语法仅需 [^key] 和 [^key]: 内容 两种格式。 ↩︎

  6. 技术博客中常用脚注来标注参考文献或技术细节。 ↩︎

  7. 学术笔记中脚注可以用来添加补充说明或引用来源。 ↩︎
